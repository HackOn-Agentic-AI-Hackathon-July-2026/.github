# HackOn Agentic AI Hackathon

Welcome to the official GitHub organization for HackOn Agentic AI Hackathon!

## 🚀 Getting Started

1. Accept your organization invitation (check your email)
2. Use our template to create your project repository
3. Follow security guidelines to protect your credentials

## ✉️ Step 1 — Accept Your GitHub Invitation

At the start of the hackathon you will receive an email from `noreply@github.com` inviting you to join this organization.

1. Open the email and click the invitation link
2. Sign in to GitHub (or create a free account at [github.com](https://github.com) if you do not have one)
3. Click **Join** to accept the invitation

> No invitation email? Check your spam folder, then contact the support team.

## 📋 Step 2 — Create Your Project Repository

1. Go to: [hackathon-project-template](https://github.com/ORG-NAME/hackathon-project-template)
2. Click "Use this template"
3. Click "Create a new repository"
4. Set **Owner** to this organization (not your personal account)
5. Name your repo using the pattern: `team-name-project-name`
   - Examples: `team-alpha-chatbot`, `team-beta-analytics-dashboard`
   - Use lowercase letters, numbers, and hyphens only — no spaces
6. Set visibility to **Private**
7. Click "Create repository"

## 📦 How to Submit Your Code Repository

When filling in the hackathon deliverables form, follow these steps to get your repository link:

1. Go to your repository: `https://github.com/ORG-NAME/team-name-project-name`
2. Copy the full URL from your browser's address bar
3. Paste it into the **Code repository** field on the submission form

**Before you submit — check all of these:**
- [ ] You accepted the GitHub org invitation
- [ ] Repository was created from `hackathon-project-template` (not from scratch)
- [ ] Repository owner is this organization, not your personal account
- [ ] Repository name follows the `team-name-project-name` pattern
- [ ] Repository visibility is **Private**
- [ ] Repository URL is copied and ready to paste into the submission form

> Need detailed instructions? See the [Participant Code Submission Guide](../../Participant_Code_Submission_Guide.md).

## 🔒 Security Guidelines

- Never commit credentials or API keys
- Use environment variables (`.env` file)
- Review the template's `SECURITY.md` file
- Ask for help if unsure!

Contact hackathon support team for assistance.
